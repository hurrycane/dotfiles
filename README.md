# Bogdan Gaza's Dotfile
These is a collection of my dot files.

## Usage

```
./install.sh # inspect output
./install.sh | sh

# Init submodules
git submodule update --init --recursive

# Update submodules to latest (from time to time)
git submodule update --remote --merge
```
